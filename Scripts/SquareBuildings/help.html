Create Square Building

Title Create Square Building

Summary

Creates square buildings from linear features.

Assumptions:

  • Line features are projected with units of meters.

  • Offset attribute is called "Width_Ft" and is a number in feet

  • There is a field called "Right_Left" with "Right" or "Left" in it.

If your data does not meet these assumptions, consider using the "Line To Rectangle" tool.


Usage

See the summary

Syntax

SquareBuilding_AlaskaPak (Building_Edges, Building_Polygons)

Parameter Explanation Data Type
Building_Edges Dialog Reference

The input polyline feature that must meet the following requirements:

  • Each feature represents 1 edge of a building.

  • The lines are projected in meters.

  • The building width is in a field called "Width_Ft" and is a number in feet

  • There is a field called "Right_Left" with "Right" or "Left" in it. This designates the location of the building in relation to a person walking from the start of the line to the end of the line.

  • There is no field called "zz_offset" in the line feature. (this will be used to hold a temporary variable)
