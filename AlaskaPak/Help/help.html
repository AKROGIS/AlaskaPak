Alaska Pak Help

This is the help document for version 3.0 of Alaska Pak. This version is built for ArcGIS 10, and will not run on previous versions of ArcGIS. It is only available as an Add-in (see ArcGIS help for using Add-Ins).

This add-in is a work in progress, and will be updated on a regular basis. We are expecting Alaska Pak 3.0 to be complete in the Spring of 2011.

Several of the commands are being proxied by the similar built-in command or toolbox. In some cases the Alaska Pak commands were built because this built-in functionality did not exist in previous versions, and the new built-in command is now the prefered method, and the olde Alaska Pak command will not be replicated.

However, in some cases the Alaska Pak provided additional features, or simplied a complex tool in which case the Alaska Pak tool be be made available to replace the proxy.

Known Issues

  1. Some tools (i.e. Add XY Coordinates, Random Selection, Copy Raster Symbology) display a picklist of layers that the user can use in the tool. This list updates automatically whenever the user makes changes to the map's table of contents even if the tool is open. However if there are multiple data frames in the map, only one data frame is active at a time, and while you can make changes to any data frame, the tools will only recognize changes to the active data frame. If you make changes to a non-active dataframe, simply make it active, and the tool will recognize the changes and update the layers accordingly
  2. Most tools only require an ArcView license level, and no extensions. The following exceptions are:
    • Add Attributes:Add Globally Unique Id (ArcEditor)
    • Grids:Generate Grid Points (ArcInfo)
    • Conversions:Polyline/Polygon To Point (ArcInfo)
    • Conversions:Point To Polygon (ArcInfo)
    • Randomize:Random Points (ArcInfo or ArcView with Spatial Analyst or ArcView with 3D Analyst)
    • Randomize:Obscure Points (ArcInfo, only if using optional No Go or Must Go areas)
